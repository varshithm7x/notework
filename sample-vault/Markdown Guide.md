# Markdown Guide

Notework supports full **GitHub Flavored Markdown** (GFM). Here's a quick reference.

## Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
```

## Text Formatting

- **Bold**: `**text**`
- *Italic*: `*text*`
- ~~Strikethrough~~: `~~text~~`
- `Code`: `` `code` ``

## Lists

### Unordered
- Item 1
- Item 2
  - Nested item

### Ordered
1. First
2. Second
3. Third

### Task Lists
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task

## Links

- External: [Notework](https://example.com)
- Internal: [[Welcome]]
- With alias: [[Getting Started|Getting Started Guide]]

## Code Blocks

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
